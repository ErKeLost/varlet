| `swatches-layout` | 是否展示颜色样本                             | _boolean_                 | `true`      |
| `elevation`       | 指定应用于构件的高程，范围在0到24之间。      | _number / string_         | `undefined` |
| `canvas-layout`   | 是否展示颜色样本                             | _boolean_                 | `false`     |
| `input-layout`    | 是否展示input                                | _boolean_                 | `false`     |
